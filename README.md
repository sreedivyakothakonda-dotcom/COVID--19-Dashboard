# 🦠 Impact of COVID-19 on Students Dashboard

An interactive Power BI dashboard analyzing how the COVID-19 pandemic affected students' daily habits, mental health, stress levels, and online learning experiences.

![COVID-19 Impact Dashboard](covid19_dashboard.png)

---

## 📊 Overview

This dashboard explores survey data collected from students during the COVID-19 pandemic, visualizing the impact on time spent on fitness, online classes, social media, sleep, and stress management behaviors.

---

## 📌 Key KPIs

| Metric | Value |
|---|---|
| **Avg Time Spent on Fitness** | 0.74 hrs/day |
| **Avg Time Spent on Online Classes** | 3.21 hrs/day |
| **Avg Time Spent on Social Media** | 2.37 hrs/day |
| **Avg Time Spent on Sleep** | 7.87 hrs/day |

---

## 📈 Visuals Included

| Visual | Description |
|---|---|
| **Online Class Experience (Bar Chart)** | Distribution of student ratings — Very Poor, Average, Good, Excellent, Poor |
| **Stress Busters (Horizontal Bar)** | Top activities used to cope: Listening to music (276), Online gaming (175), Watching web series (102), Reading books (77) |
| **Social Media Time by Platform** | Bar chart comparing average hours spent on TikTok, Snapchat, Twitter, Instagram, LinkedIn, WhatsApp, etc. |
| **Avg Age & Social Media Time by Platform** | Bar chart comparing student age vs. time spent per platform |
| **Online Class Time by Medium** | Treemap showing device usage — Laptop/Desktop vs. Smartphone vs. Tablet |
| **Self Study Time by Region** | Pie chart — Outside Delhi (61.4%) vs. Delhi-NCR (38.6%) |
| **Weight Change Slicer** | Dropdown filter by weight change category |

---

## 🗂️ Dataset

- **Source:** Student survey data collected during COVID-19 lockdown
- **Fields used:** `Student ID`, `Region of residence`, `Online class experience`, `Stress busters`, `Social media platform`, `Time spent on social media`, `Time spent on online classes`, `Time spent on fitness`, `Sleep hours`, `Medium for online class`, `Weight change`

---

## 🛠️ Tools Used

- **Power BI Desktop** — Dashboard creation and data modeling
- **Power Query** — Data cleaning and transformation
- **DAX** — Calculated averages and aggregations

---

## 🚀 How to Use

1. Clone or download this repository
2. Open the `.pbix` file in Power BI Desktop
3. Use the **Weight Change** dropdown slicer to filter by student weight change category
4. Hover over visuals for detailed breakdowns

---

## 💡 Key Insights

- Most students rated their online class experience as **Average** or **Very Poor**
- **Listening to music** was the most popular stress-relief activity among students
- Students outside Delhi-NCR spent more time on self-study than those within the city
- **Laptop/Desktop** was the dominant medium for attending online classes
- Average sleep time of ~7.87 hrs suggests students maintained reasonable sleep during lockdown
